# AOS Semantic Vocabulary

Version: 1.0

Status: Core

---

# Purpose

The Vocabulary Layer translates domain-specific language into AOS concepts.

It enables different domains to share one analytical language.

The Vocabulary Layer never performs reasoning.

Its only responsibility is semantic mapping.

---

# Semantic Principle

Different words

may describe

the same abstract object.

Always translate concrete language

into ontology

before reasoning.
